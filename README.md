# ❤️ devTinder

A full-stack social matching platform inspired by Tinder, built specifically for developers. Users can discover other developers, send connection requests, and chat in real-time after matching.

## 🚀 Live Demo

🔗 http://13.200.233.99/

## 📌 Backend Repository

🔗 https://github.com/ShreySrivastav1/devTinder

---

## ✨ Features

### Authentication

- User Signup & Login
- JWT Authentication
- HttpOnly Cookie Sessions

### User Features

- Create and Edit Profile
- Discover Developers
- Send Connection Requests
- Accept / Reject Requests
- View Connections
- Real-Time Chat

### Chat Features

- One-to-One Messaging
- Socket.IO Integration
- Instant Message Delivery
- Persistent Chat History

---

## 🛠 Tech Stack

### Frontend

- React.js

### Backend

- Node.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ShreySrivastav1/devTinder-web.git
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

```env
VITE_BASE_URL=http://localhost:3000
```

### Run Application

```bash
npm run dev
```
